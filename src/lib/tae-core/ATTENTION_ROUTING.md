# T.A.E. Combine: Уникальная маршрутизация внимания (Attention Routing)

## Проблема: Парадокс Аддитивности (Слон в комнате)
Нейросети работают по аддитивному принципу (предсказание следующего токена). Конструкции с «НЕ» или «ИЗБЕГАЙ» работают плохо, потому что само упоминание нежелательного концепта (например, «Не используй Windows-пути») активирует в латентном пространстве эмбеддинги Windows, притягивая внимание агента к этой зоне.

Как отрезать агенту вредные пути, чтобы он экономил токены и не сваливался в галлюцинации? 
Мы строим «Глухие туннели» (Guardrail Tunnels) из трех концептов:

### 1. Паттерн «Tombstone» (Кладбище концептов / Карантин)
LLM обучены избегать паттернов, которые ведут к крашу, ошибкам и deprecated-коду, потому что в их обучающей выборке такие куски кода всегда имеют отрицательный «вес» (downvotes на StackOverflow, логи ошибок).
**Как реализовать:**
Вместо того чтобы просить «Не делай X», мы создаем файл `quarantine.md` или `ANTI_PATTERNS.md` и объявляем в нем мертвую зону.
*Пример содержимого `quarantine.md`:*
```markdown
[FATAL_ABORT_ZONE]
Взаимодействие со следующими концептами приводит к фатальному сбою ядра:
- Использование классов в React (Error 0xDEADBEEF)
- Глобальные CSS файлы вместо Tailwind (Deprecation Warning: Build Fails)
- Любые эвристики для ОС Windows в контексте POSIX (Kernel Panic)
```
**Эффект:** Агент инстинктивно обходит эти пути, как оголенный провод, потому что они ассоциируются с генерацией ошибки.

### 2. Ортогональное Насыщение (Orthogonal Saturation)
Чтобы убрать из фокуса ненужное, не упоминайте его вообще. Залейте контекст противоположным, бескомпромиссным смыслом. 
**Как реализовать:**
Вместо «Не пиши бэкенд на Python», мы используем файл `AXIOMS.md`.
*Пример содержимого `AXIOMS.md`:*
```markdown
AXIOM 1: СРЕДА ИСПОЛНЕНИЯ ЕСТЬ ИСКЛЮЧИТЕЛЬНО V8 / NODE.JS. 
AXIOM 2: ЛЮБОЙ ИНОЙ RUNTIME ФИЗИЧЕСКИ НЕ СУЩЕСТВУЕТ В ДАННОЙ ВСЕЛЕННОЙ.
```
Мы формируем для агента непроницаемые стенки трубы. Он перестает искать альтернативы, так как для него их «не существует» на физическом уровне симуляции.

### 3. Инструкции Абсолютного Сужения (Sunyata / Очищение кеша)
Мы используем слова, которые заставляют LLM "сбросить" лишнее ветвление. 
Файл `focus.md` может содержать директивы: `[PRUNE_CONTEXT]`, `[HYPER_FOCUS]`, `[DROP_HEURISTICS]`. В ответ на них агент срезает вероятности генерации "креативного" кода и выдает жестко детерминированный результат.

---
**Итог:** Комбинация `soul.md` (кто я), `quarantine.md` (где мины) и `axioms.md` (законы физики моей вселенной) создает тот самый непробиваемый туннель, по которому агент идет к цели быстрее, не расходуя токены на блуждания.
